# ManyChat SEO Empire 2026 🚀

An automated, programmatic SEO engine designed to capture high-intent affiliate traffic for ManyChat. Built for GitHub Pages deployment with daily auto-indexing.

## 📊 Performance Metrics
- **Target:** 1,000 High-Intent Keywords
- **Frequency:** 20 Unique Pages per Day
- **Optimization:** Bing 2026 Rich Snippet Logic
- **Architecture:** Zero-Duplicate Python Generation

## 📂 Core Components
- `/manychat/`: Target directory for all 1,000 generated HTML assets.
- `manychat-gen.py`: The content engine. Uses modular templates to build unique SEO pages.
- `keywords.txt`: The master database. Contains 1,000 keywords siloed by profitability.
- `blog.html`: The category hub. Directs crawlers to the newest generated content.
- `.github/workflows/`: Automation scripts that handle daily generation and indexing.

## 🛠️ Installation & Setup
1. **Initialize Keywords:** Run the keyword injector or manually populate `keywords.txt`.
2. **Update Links:** Replace the affiliate URLs in `manychat-gen.py` with your PartnerStack links.
3. **Deploy:** Push to GitHub. The Actions workflow will trigger automatically at Midnight UTC.

## 🛡️ SEO Guardrails
- **Anti-Duplication:** Logic built-in to skip existing filenames.
- **Freshness Injection:** Every page receives 3-5 random "2026 Trend Phrases" to maintain high indexing priority.
- **Mobile-First CSS:** All generated pages are optimized for modern Core Web Vitals.
